# 🌺 宮古島防衛戦線 〜まもるくん VS スーパー台風〜 要件定義・仕様書

## 1. プロジェクト概要
- **タイトル（仮）**: 宮古島防衛戦線 〜まもるくん VS スーパー台風〜
- **ジャンル**: 拠点防衛型 タワーディフェンス（NIKKE「FOR THE KING」/ Kingdom Rush ライク）
- **コンセプト**: 宮古島に上陸し、集落へ向かって進行してくる巨大台風と暴風雨。プレイヤーは道の脇の要所に「宮古島まもるくん」を配置・強化し、島民の平和を守り抜く。
- **ターゲット**: 全年齢・カジュアル〜コアゲーマー・宮古島ファン
- **プラットフォーム**: PC / スマートフォン両対応のWebブラウザ（インストール不要）

## 2. 技術スタック（システム要件）
2Dブラウザゲーム開発の世界的スタンダードである「Phaser 3」を採用します。

- **ゲームエンジン**: **Phaser 3** (JavaScript 2D Game Framework)
  - **Path（経路）機能**: Phaser内蔵の `Phaser.Curves.Path` と `PathFollower` を使えば、うねる道を敵が自動で歩く処理が数行で実装可能。
  - **索敵と攻撃**: タワーの射程（円形）に入った敵を距離計算で捕捉し、弾を飛ばす処理もPhaserの標準機能と相性抜群です。
- **コア言語**: JavaScript (ES2022+) または TypeScript
- **描画**: WebGL（非対応ブラウザはCanvasへフォールバック）。弾幕や敵が大量に出てもサクサク動きます。
- **開発・公開環境**: Vite等でのローカル開発、または単一HTMLでのモック開発 / 公開は GitHub Pages や Vercel 等。

## 3. ゲームシステム仕様

### 3.1. 基本ルール・進行
- **マップ構成**: 画面端の「敵の出現地点（海）」から「防衛拠点（集落）」まで続く曲がりくねった**道（ルート）**が存在。
- **Wave（ウェーブ）制**: 一定時間ごとに敵が集団（Wave）となって道に沿って攻めてくる。全Waveをしのぎ切ればステージクリア。
- **敗北条件**: 敵が防衛拠点（ゴール）に到達すると拠点のライフが減り、`0` になるとゲームオーバー。

### 3.2. 配置とアップグレード（まもるくんの運用）
- **コスト（シマコイン）**: 敵を倒すとお金（シマコイン）を獲得できる。
- **配置（召喚）**: 
  - 道の周辺にある「配置可能スロット（台座）」をクリックする。
  - 画面下部のメニューから任意のまもるくんを選び、コストを消費して配置する。
- **自動迎撃**: 配置されたまもるくんは移動せず、自身の射程（円形）内に敵が入ると自動で攻撃（射撃など）を行い続ける。
- **アップグレード**: 配置済みのまもるくんをクリックし、コインを消費して「レベルアップ（攻撃力や射程の増加）」または「撤去（コインを一部返還）」が可能。

### 3.3. アクティブスキル（バーストスキル機能）
- NIKKEの「FOR THE KING」の醍醐味である、プレイヤーの直接介入要素。
- 画面下部のゲージが時間経過（または敵撃破）で溜まると、下部のまもるくんアイコンをマップ上の任意の場所に**ドラッグ＆ドロップ**できるようになり、強力な必殺技（指定範囲への大ダメージや、一定時間の敵の移動速度低下など）を発動できる。

## 4. キャラクター仕様（後日画像差し替え）
ご用意いただく「5人のまもるくん画像」を、タワーディフェンスの王道ロール（役割）に当てはめます。Phaserの `preload` 機能でファイル名を指定するだけで簡単にゲームへ反映される作りにします。

| No | 役割（TD的ロール） | コスト | 攻撃の特徴 | NIKKE (FTK) での相当枠 |
|:---|:---|:---|:---|:---|
| 1 | **連射まもるくん** | 低 | 射程は短いが、機関銃のように高速で敵単体を攻撃。序盤の主力。 | タレット / マシンガン系 |
| 2 | **大砲まもるくん** | 中 | 攻撃速度は遅いが、着弾点で爆発し複数の敵をまとめて攻撃（範囲）。 | 大砲 / モダニア（範囲） |
| 3 | **足止めまもるくん**| 中 | 攻撃力は低いが、攻撃を当てた敵の移動速度をしばらく下げる。 | 氷の塔 / クラウン（減速） |
| 4 | **狙撃まもるくん** | 高 | 攻撃間隔は非常に長いが、超長射程・超高火力で硬い敵を撃ち抜く。 | スノーホワイト |
| 5 | **司令塔まもるくん**| 高 | 自身は攻撃せず、周囲に配置されたまもるくんの攻撃力や連速をUPする。 | キロ（周囲バフ） |

## 5. 敵軍ユニット（台風軍団）
道に沿って歩いてくる自然の脅威。ステータスでバリエーションを持たせます。
- **雨粒（ザコ）**: 移動速度・HPともに普通。数が多い。
- **突風（スピード）**: 移動速度が極めて速く、防衛線をすり抜けやすい。（足止めまもるくんが有効）
- **雷雲（タンク）**: HPが非常に高く、なかなか倒れない。（狙撃まもるくんが有効）
- **スーパー台風（ボス）**: 最終Waveに登場する超巨大な敵。圧倒的なHPを持つため、全まもるくんのスキルを駆使して総力戦で挑む。

Phase 1: コアサイクルのプロトタイプ開発（白箱）
